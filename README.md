# Trophy Case Sports Shopping Cart

## Overview
This repository contains an e-commerce shopping cart website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website simulates an online sports memorabilia store and demonstrates JavaScript programming concepts related to shopping cart functionality, order processing, dynamic table generation, price calculations, and customer purchase summaries.

## Features
* E-commerce shopping cart interface
* Dynamic shopping cart generation
* Automatic subtotal calculation
* Product quantity tracking
* Order summary display
* JavaScript-generated HTML tables
* Sports memorabilia product catalog
* Responsive page layout

## Technologies Used
* HTML5
* CSS3
* JavaScript
* Dynamic HTML Generation
* Arrays and Loops

## Project Files
* `tc_cart.html` – Main shopping cart webpage
* `tc_cart.js` – Shopping cart generation and subtotal calculations
* `tc_order.js` – Product inventory and order data
* `tc_reset.css` – CSS reset stylesheet
* `tc_styles.css` – Layout and visual styling
* Product image and branding assets

## Skills Demonstrated
* Front-end web development
* JavaScript programming
* Dynamic HTML generation
* Array manipulation
* Loop processing
* Price and subtotal calculations
* Data organization
* E-commerce interface design

This project was completed as part of web development coursework and demonstrates JavaScript programming concepts used in e-commerce applications, including shopping cart management, order calculations, and dynamic webpage content generation.
